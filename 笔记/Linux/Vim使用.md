# Vim 使用



### 光标移动

| ⬅⬆➡⬇      | 光标向左、上、右、下移动一个字符                             |
| --------- | ------------------------------------------------------------ |
| Home      | 移动到行首                                                   |
| End       | 移动到行尾                                                   |
| Page Up   | 屏幕向上移动一页                                             |
| Page Down | 屏幕向下移动一页                                             |
| H         | 光标移动到这个屏幕的最上面那一行的第一个字符                 |
| M         | 光标移动到这个屏幕的中央那一行的第一个字符                   |
| L         | 光标移动到这个屏幕的最下面那一行的第一个字符                 |
| G         | 移动到这个文档的最后一行                                     |
| nG        | n为数字。移动到这个文档的第n行。例如20G会移动到这个档案的第20行（可配合 :set nu） |
| gg        | 移动到档案的第一行，相当于1G                                 |
| n<Enter>  | n为数字。光标向下移动n行                                     |
| ctrl + o  |                                                              |
|           |                                                              |
